## Context

本项目是一个货币基金ETF折价套利监控系统，专注于银华日利(511880)和华宝添益(511990)的买入赎回套利机会。需要部署在免费云平台上持续运行监控。

**约束条件：**
- 免费云端部署（资源有限制）
- 需要实时行情数据源（买1/卖1价格）
- 基金净值T+1公布，可用年化1%估算当日净值
- 买入赎回T+1到账，需考虑交易日（排除周末和长假）
- 交易成本为0

## Goals / Non-Goals

**Goals:**
- 实时获取买1价格、卖1价格
- 用前一交易日净值估算当日净值
- 准确计算卖1价格买入赎回的年化收益率
- 计算实际到账天数（考虑非交易日）
- 交易日9:30-15:00，每5分钟发送一次通知
- 当年化收益率超过阈值时通知用户
- 提供 Web 界面实时展示数据
- 支持免费云平台部署

**Non-Goals:**
- 不做申购卖出套利计算
- 不存储历史记录
- 不做交易执行，仅监控提醒
- 不支持其他基金

## Decisions

### 数据源选择
- **行情数据**: 使用东方财富/新浪财经等免费行情接口，获取实时买1/卖1价格
- **净值数据**: 从天天基金等平台获取最近公布的净值

### 当日净值估算
```
当日估算净值 = 最近公布净值 × (1 + 1% / 365 × 天数)
```

### 交易日判断与到账天数
- T日买入，T+1日赎回款到账
- T+1必须是交易日，遇到周末/长假顺延
- 需要交易日历或简单规则判断

**年化收益率计算：**
```
折价率 = (估算净值 - 卖1价格) / 估算净值
实际占用天数 = 到账日期 - 买入日期（自然日）
年化收益率 = 折价率 × 365 / 实际占用天数
```

### 通知时机
- **通知窗口**: 仅在交易日 9:30-15:00 发送通知
- **通知频率**: 每5分钟发送一次
- **非交易时间**: 不发送通知

### 技术栈
- **后端**: Python + FastAPI
- **定时任务**: APScheduler
- **数据存储**: 内存缓存
- **前端**: 单页 HTML + JavaScript
- **通知**: 微信（Bark/Server酱）、邮件
- **交易日历**: 内置简单规则或调用交易所日历API

### 免费部署
推荐 Render 或 Railway

## Risks / Trade-offs

| 风险 | 缓解措施 |
|------|----------|
| 净值估算误差 | 货币基金收益稳定，误差可控 |
| 交易日判断 | 使用交易所日历或内置规则 |
| 数据源不稳定 | 多数据源备份 |
| 免费平台休眠 | 选择不休眠平台或定时唤醒 |
