## Why

银华日利(511880)和华宝添益(511990)是场内货币基金ETF，当二级市场价格低于净值时存在折价套利机会：二级市场买入后赎回，赚取差价。

目前缺乏工具实时计算**卖1价格买入赎回年化收益率**，投资者难以判断套利是否值得操作。需要监控年化收益率，当高于设定阈值时提醒用户。

## What Changes

- 新建银华日利、华宝添益折价套利监控程序
- 实时采集基金二级市场买1价格、卖1价格和净值数据
- 计算卖1价格买入赎回年化收益率
- 当年化收益率超过设定阈值时发送告警通知
- 提供 Web 界面展示实时监控数据
- 支持 Docker 容器化云端部署

## Capabilities

### New Capabilities

- `fund-data-collection`: 基金数据采集 - 抓取银华日利、华宝添益的买1价格、卖1价格和基金净值
- `annualized-return-analysis`: 年化收益率计算 - 计算卖1价格买入赎回年化收益率
- `alert-notification`: 告警通知 - 当年化收益率超过阈值时推送通知
- `web-dashboard`: Web 仪表盘 - 展示实时监控数据、收益率
- `cloud-deployment`: 云端部署 - Docker 容器化部署支持
